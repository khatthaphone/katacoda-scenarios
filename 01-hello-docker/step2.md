# Docker basic commands

ຣັນ Docker container
`docker run [args]`

ເບິ່ງ Container ທີ່ຣັນຢູ່
`docker ps`

ເບິ່ງ Container ທັງໝົດ
`docker ps -a`

ປິດ/ຢຸດ Container
`docker stop [containd-id/container-name]`

ລຶບ Container
`docker rm [containd-id/container-name]`

ເບິ່ງ Image
`docker images`

ລຶບ Image
`docker rmi [image-id/image-name]`
